Шаг 2. Анализ данных¶

Для начала определим классы данных:

Код штата принимает значение из определённого набора и является дискретными данными, в то время как цена за фунт (в $) и общее количество произведённого мёда может принять любое значение из промежутка и является непрерывными данными.

Посчитаем основные характеристики:

Математическое ожидание: $$M[X] = \frac{\sum_{i=1}^{n}x_i}{n}$$ или $$M[X] = \frac{\sum_{i=1}^{k}x_i^{'} n_i}{n}$$

Математическое ожидание = 21

Дисперсия: $$D[X] = \sigma^{2} = \frac{\sum_{i=1}^{n}(x_i - M[X])^{2}}{n}$$ или $$D[X] = \sigma^{2} = \frac{\sum_{i=1}^{k}(x_i^{'} - M[X])^{2}n_i}{n}$$

Определим шаг по формуле Стерджесса $$h = \frac{x_{max} - x_{min}}{1 + 3,22\lg n}$$

In [18]:
min_value = np.amin(totalprod)
max_value = np.amax(totalprod)
vrange = max_value - min_value
h = (max_value - min_value) / (1 + 3.22 * np.log10(n)) # шаг
k = int(np.floor((max_value - min_value) / h)) # количество интервалов
frequencies_dict = Counter(totalprod)
values = np.array(frequencies_dict.keys())
values.sort()
totalprod = np.array(totalprod)
