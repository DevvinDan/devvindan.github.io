Шаг 2. Анализ данных¶

Для начала определим классы данных:

Число колоний и количество всего произведённого мёда (в данном случае это всегда целое число в фунтах) являются дискретными данными, в то время как цена за фунт (в $) может принять любое значение из промежутка и является непрерывными данными.

Начнём с анализа числа колоний. Первым делом составим вариационный ряд, посчитаем размах и составим таблицу частот:

In [124]:
frequencies_dict = Counter(numcol)
values = np.array(frequencies_dict.keys())
values.sort()
frequencies = np.array([frequencies_dict[key] for key in values])
rel_frequencies = np.divide(frequencies.astype(float), n)
vrange = np.ptp(values)
print("Размах (максимальное - минимальное значение): {}".format(vrange))
